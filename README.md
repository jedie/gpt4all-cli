CLI for gpt4all

Start hacking: Just clone the sources and `ask`

e.g.:

```commandline
~$ git clone https://github.com/jedie/gpt4all-cli.git
~$ cd gpt4all-cli
~/gpt4all-cli$ ./cli.py ask What is Python?
```
